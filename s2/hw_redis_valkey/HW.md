## Задание 6 (бонус). Pub/Sub

Откройте **два** терминала с `redis-cli`.

**Терминал 1** — подписчик:

```bash
docker exec -it redis redis-cli
```

```
SUBSCRIBE news
```

**Терминал 2** — издатель:

```bash
docker exec -it redis redis-cli
```

```
PUBLISH news "Hello from Redis!"
PUBLISH news "Second message"
```
